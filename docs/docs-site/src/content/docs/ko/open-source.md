# 오픈소스 기술

이 문서 사이트는 다음 오픈소스 기술로 구축되었습니다.

## 문서 프레임워크

| 기술 | 설명 | 라이선스 |
|------|------|----------|
| [Astro](https://astro.build) | 콘텐츠 중심 웹사이트를 위한 웹 프레임워크 | MIT |
| [Starlight](https://starlight.astro.build) | Astro용 완전한 기능의 문서 테마 | MIT |

## 핵심 기술

| 기술 | 설명 | 라이선스 |
|------|------|----------|
| [TypeScript](https://www.typescriptlang.org) | 타입 문법이 있는 JavaScript | Apache-2.0 |
| [Node.js](https://nodejs.org) | JavaScript 런타임 | MIT |
| [pnpm](https://pnpm.io) | 빠르고 디스크 효율적인 패키지 매니저 | MIT |

## UI & 스타일링

| 기술 | 설명 | 라이선스 |

## 메인 모노레포 기술

문서화된 애플리케이션들은 추가 기술을 사용합니다:

| 기술 | 설명 | 라이선스 |
|------|------|----------|
| [React](https://react.dev) | 웹 및 네이티브 UI 라이브러리 | MIT |
| [React Router](https://reactrouter.com) | React용 클라이언트 사이드 라우팅 | MIT |
| [Turborepo](https://turbo.build) | 고성능 빌드 시스템 | MIT |
| [Zod](https://zod.dev) | TypeScript 우선 스키마 검증 | MIT |
| [MiniSearch](https://lucaong.github.io/minisearch/) | 클라이언트 사이드 전문 검색 | MIT |
| [Dexie.js](https://dexie.org) | IndexedDB 래퍼 | Apache-2.0 |
| [Framer Motion](https://www.framer.com/motion/) | React용 애니메이션 라이브러리 | MIT |
| [Zustand](https://zustand-demo.pmnd.rs) | React용 상태 관리 | MIT |
| [Biome](https://biomejs.dev) | 린터 및 포매터 | MIT |
| [Vitest](https://vitest.dev) | 유닛 테스팅 프레임워크 | MIT |
| [Playwright](https://playwright.dev) | E2E 테스팅 프레임워크 | Apache-2.0 |
