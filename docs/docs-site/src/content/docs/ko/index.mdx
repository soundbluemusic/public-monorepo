## 왜 SSG인가?

<CardGrid>
  <Card title="⚡ 즉시 로드" icon="rocket">
    전 세계 CDN 엣지 위치에서 미리 렌더링된 HTML 제공. 서버 처리 없음, 대기 없음.
  </Card>

  <Card title="🔒 기본 보안" icon="approve-check">
    서버가 없으면 서버 취약점도 없습니다. 정적 파일은 본질적으로 안전합니다.
  </Card>

  <Card title="💰 무료 호스팅" icon="sun">
    무제한 대역폭의 Cloudflare Pages에서 호스팅. 유지할 서버 없음.
  </Card>

  <Card title="🌐 SEO 최적화" icon="magnifier">
    검색 엔진이 사용할 수 있는 전체 HTML 콘텐츠. 적절한 메타 태그와 사이트맵 포함.
  </Card>
</CardGrid>
