# Roots — 数学ドキュメント

**学習者のための数学ドキュメント** | 920 SSGページ

Rootsは、代数、幾何学、微積分など様々な数学分野を体系的に整理した数学ドキュメントサイトです。

## ライブデモ

🌐 **[roots.soundbluemusic.com](https://roots.soundbluemusic.com)**

## 機能

### 📐 体系的な数学概念

- **438概念** - 基礎から上級まで
- **18分野** - 代数、幾何学、微積分など
- 前提知識の概念接続
- 数式と証明

### 🔗 概念間の接続

- 前提知識の概念表示
- 関連概念リンク
- 学習経路の提案

### 🌐 多言語サポート

- 英語と韓国語UI
- 数学用語のバイリンガル表示

## プロジェクト構造

```
apps/roots/
├── app/
│   ├── components/      # Reactコンポーネント
│   ├── routes/          # React Routerルート
│   ├── data/            # 数学概念データ
│   └── utils/           # ユーティリティ関数
├── public/              # 静的アセット
└── react-router.config.ts  # SSG設定
```

## 主要ルート

| ルート | 説明 |
|--------|------|
| `/` | ホームページ |
| `/concept/:conceptId` | 個別概念ページ（英語） |
| `/ko/concept/:conceptId` | 個別概念ページ（韓国語） |
| `/field/:fieldId` | 分野別概念一覧 |
| `/about` | 概要ページ |

## 数学分野

| 分野 | 概念数 | 説明 |
|------|--------|------|
| 代数 | 85 | 方程式、不等式、関数 |
| 幾何学 | 72 | 図形、座標、変換 |
| 微積分 | 64 | 極限、微分、積分 |
| 線形代数 | 48 | ベクトル、行列、線形変換 |
| 確率統計 | 56 | 確率、分布、推定 |
| その他 | 113 | 論理、集合、数論など |
