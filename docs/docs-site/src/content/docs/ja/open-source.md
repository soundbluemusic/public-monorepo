# オープンソース技術

このドキュメントサイトは以下のオープンソース技術で構築されています。

## ドキュメントフレームワーク

| 技術 | 説明 | ライセンス |
|------|------|----------|
| [Astro](https://astro.build) | コンテンツ駆動型ウェブサイト向けウェブフレームワーク | MIT |
| [Starlight](https://starlight.astro.build) | Astro用フル機能ドキュメントテーマ | MIT |

## コア技術

| 技術 | 説明 | ライセンス |
|------|------|----------|
| [TypeScript](https://www.typescriptlang.org) | 型構文を持つJavaScript | Apache-2.0 |
| [Node.js](https://nodejs.org) | JavaScriptランタイム | MIT |
| [pnpm](https://pnpm.io) | 高速でディスク効率の良いパッケージマネージャー | MIT |

## UI & スタイリング

| 技術 | 説明 | ライセンス |
|------|------|----------|

| [Shiki](https://shiki.style) | シンタックスハイライター | MIT |
| [Expressive Code](https://expressive-code.com) | Astro用コードブロック | MIT |
