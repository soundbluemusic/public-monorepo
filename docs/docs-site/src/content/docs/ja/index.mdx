## なぜSSG？

<CardGrid>
  <Card title="⚡ 即時ロード" icon="rocket">
    世界中のCDNエッジロケーションから事前レンダリングされたHTMLを配信。サーバー処理なし、待機なし。
  </Card>

  <Card title="🔒 デフォルトでセキュア" icon="approve-check">
    サーバーがなければサーバーの脆弱性もありません。静的ファイルは本質的に安全です。
  </Card>

  <Card title="💰 ゼロコストホスティング" icon="sun">
    無制限帯域幅のCloudflare Pagesでホスティング。維持するサーバーなし。
  </Card>

  <Card title="🌐 SEO最適化" icon="magnifier">
    検索エンジンが利用できる完全なHTMLコンテンツ。適切なメタタグとサイトマップ含む。
  </Card>
</CardGrid>
