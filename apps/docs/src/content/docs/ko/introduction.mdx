## 이것은 무엇인가요?

Cloudflare Workers에서 호스팅되는 세 가지 학습 앱을 포함하는 모노레포입니다:

- **Context** — 학습자를 위한 한국어 사전
- **Permissive** — 웹 개발 리소스
- **Roots** — 수학 문서

모든 앱은 SEO 최적화를 위해 **서버 사이드 렌더링(SSR)**을 사용합니다. SPA 모드 없음.

## 기술 스택

| 카테고리 | 기술 |
|:---------|:-----|
| 프레임워크 | React 19 + TanStack Start |
| 언어 | TypeScript 5.x |
| 스타일링 | Tailwind CSS v4 |
| 데이터베이스 | Cloudflare D1 |
| 검색 | MiniSearch |
| i18n | Paraglide |
| 호스팅 | Cloudflare Workers |
| 빌드 | Vite + Turborepo |
