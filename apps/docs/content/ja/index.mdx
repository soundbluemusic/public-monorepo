# SoundBlue Public Monorepo

学習者のための3つのアプリケーション — 韓国語辞書 (SSR + D1)、Web開発リソース (SSR)、数学ドキュメント (SSR)

[はじめる](/public-monorepo/ja/guides/introduction/) | [GitHubで見る](https://github.com/soundbluemusic/public-monorepo)

## アプリケーション

| App | Description | Mode |
|-----|-------------|------|
| **Context** | 学習者のための韓国語辞書 | SSR + D1 |
| **Permissive** | Web開発リソース | SSR |
| **Roots** | 数学ドキュメント | SSR |

---

## レンダリングモード

### ⚡ SSR + D1 (Context)
Cloudflare D1データベースを活用したサーバーサイドレンダリング。16,836エントリを動的に提供し、即時更新されます。

### 🔒 SSR (Permissive)
Web開発リソースのためのサーバーサイドレンダリング。高速で安全、SEOに最適化されています。

### 📐 SSR (Roots)
数学ドキュメントのためのサーバーサイドレンダリング。すべてのページがCloudflare Workersで動的にレンダリングされます。

### 🌐 SEO最適化
検索エンジンが利用できる完全なHTMLコンテンツ。すべてのモードでクローラーに完全なHTMLを提供します。

## 技術スタック

- **React Router v7** - 動的コンテンツのためのD1連携SSRモード
- **TypeScript** - 10の共有パッケージと3つのアプリケーション全体にわたる厳格な型安全性
- **Turborepo** - 97%以上のキャッシュヒット率を持つモノレポビルドオーケストレーション
- **Tailwind CSS v4** - カスタムデザイントークンを使用したユーティリティファーストスタイリング
