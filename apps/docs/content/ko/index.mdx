# SoundBlue Public Monorepo

학습자를 위한 3개의 애플리케이션 — 한국어 사전 (SSR + D1), 웹 개발 리소스 (SSR), 수학 문서 (SSR)

[시작하기](/public-monorepo/ko/guides/introduction/) | [GitHub에서 보기](https://github.com/soundbluemusic/public-monorepo)

---

## 애플리케이션

| App | Description | Mode |
|-----|-------------|------|
| **Context** | 학습자를 위한 한국어 사전 | SSR + D1 |
| **Permissive** | 웹 개발 리소스 | SSR |
| **Roots** | 수학 문서 | SSR |

---

## 렌더링 모드

### ⚡ SSR + D1 (Context)
Cloudflare D1 데이터베이스를 활용한 서버 사이드 렌더링. 16,836개 엔트리를 동적으로 제공하며 즉시 업데이트됩니다.

### 🔒 SSR (Permissive)
웹 개발 리소스를 위한 서버 사이드 렌더링. 빠르고 안전하며 SEO에 최적화되어 있습니다.

### 📐 SSR (Roots)
수학 문서를 위한 서버 사이드 렌더링. 모든 페이지가 Cloudflare Workers에서 동적으로 렌더링됩니다.

### 🌐 SEO 최적화
검색 엔진이 사용할 수 있는 전체 HTML 콘텐츠. 모든 모드에서 크롤러에게 완전한 HTML을 제공합니다.

---

## 기술 스택

- **React Router v7** - 동적 콘텐츠를 위한 D1 연동 SSR 모드
- **TypeScript** - 10개의 공유 패키지와 3개의 애플리케이션 전체에 걸친 엄격한 타입 안전성
- **Turborepo** - 97% 이상의 캐시 적중률을 가진 모노레포 빌드 오케스트레이션
- **Tailwind CSS v4** - 커스텀 디자인 토큰을 사용한 유틸리티 우선 스타일링
